# What is this?

This is the code for my personal website, which uses my own static site generator,
[sugo](https://github.com/lmihailovic/sugo).

## How to deploy

Go to the root of the repository and run:

```shell
sugo
```

Then, copy the contents of the `website/` directory to your server's `public_html` directory.
